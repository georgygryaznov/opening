# Система управленческой отчётности — ГК «Открытие»

Репозиторий для разработки системы управленческой отчётности группы предприятий «Открытие».

## Стадия проекта
**Presale** — сбор и согласование требований (BRD).

## Структура репозитория

| Папка | Назначение |
|---|---|
| `BRD/` | Business Requirements Document — описание бизнеса и список отчётов |
| `docs/` | Техническая документация (будет наполняться на этапе разработки) |
| `src/` | Исходный код (будет наполняться на этапе разработки) |

Подробнее — в [BRD/README.md](BRD/README.md).
